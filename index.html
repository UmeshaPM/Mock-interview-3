1. Difference between flex row and column

Flex row-It is the type of CSS flex direction property in which the items are align in row manner. It display items on horizontal axis in row manner. We can reverse the row order in flex.
Syntax:

div {
            display: flex;
            flex-direction: row;
        }


Flex column-Flex column is also the type of CSS flex direction property in which the items are align in column manner.It display items on vertical axis in column manner. Similar to row, we can reverse the column order in flex.
Syntax:

div {
            display: flex;
            flex-direction: column;
        }

2.Explain inline,internal and external stylesheet

Inline CSS- It allows you to add styles to the specific HTML element in between any lines.
Internal CSS- It allows you to add styles inside the style tag within head section of html.
External CSS- It allows you to write all the CSS code in a css file.It works by linking a .css file containing all the CSS rules to a HTML elemnts.For example Style.css.

6.What is z-index in CSS

The z-index property specifies the stack order of an element.
z-index only works on positioned elements like (position: absolute, position: relative, position: fixed, or position: sticky) and It is used to define the order of elements if they overlap on each other.


7.What is the difference between padding and margin

Margin: It is the space around an element. Margins are used to move an element up or down on a page as well as left or right.
Margin is completely transparent
The CSS margin properties are used to create space around elements, outside of any of any defined borders.

Padding: It is the space between the element and the related content inside it.
Padding can be affected by background colors as it covers the area around the content.
The CSS padding properties are used to generate space around an element's content, inside of any defined borders
